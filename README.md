


# منصة العميد التعليمية - El-Amid Educational Platform

منصة متكاملة لأستاذ الرياضيات (العميد) للمرحلة الثانوية.

## التقنيات المستخدمة
- **الباك إند**: Node.js, Express, MongoDB, JWT
- **الفرونت إند**: Next.js (App Router), Vanilla CSS, Framer Motion
- **الفيديوهات**: Dailymotion Player API

## المميزات
- تسجيل دخول بكود الطالب (كود واحد لجهاز واحد).
- تقسيم تلقائي للمحتوى حسب الصف الدراسي.
- نظام امتحانات أونلاين بتصحيح تلقائي وتوقيت محدد.
- لوحة تحكم أدمن كاملة لإدارة الطلاب والمحتوى.
- حماية المحتوى من التحميل.

## طريقة التشغيل

### 1. تشغيل الباك إند (Back-End)
```bash
cd backend
npm install
npm run dev
```
*سيقوم الباك إند بالعمل على `http://localhost:5000`*

### 2. تشغيل الفرونت إند (Front-End)
```bash
cd frontend
npm install
npm run dev
```
*سيقوم الفرونت إند بالعمل على `http://localhost:3000`*

## التحسينات الأخيرة (Recent Tuning)
- تم إضافة نظام `api utility` مركزي لتسهيل التعامل مع الـ Endpoints.
- تم ضبط `Next.js config` و `jsconfig` لدعم مسارات الـ alias (@/).
- تم تجهيز ملفات بيئة العمل (.env) لكل من الباك والفرونت.

## ملاحظات أمنية
- النظام يمنع دخول نفس الكود من جهازين في وقت واحد.
- يتم تخطى حواجز المتصفح لعرض الفيديوهات داخل المنصة فقط.
- لوحة الأدمن محمية بصلاحيات خاصة.
